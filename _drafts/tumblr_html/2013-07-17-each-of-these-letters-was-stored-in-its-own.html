Each of these letters was stored in its own shallow drawer in a case, so the first task for an apprentice was to learn which drawer each letter was stored in. This was known as ‘learning the case’. Some cases contained only lowercase letters, with the capital letters stored in another — from which the words ‘lowercase’ and ‘uppercase’ came.
—Ever wondered where some of the words we use every day come from? Fascinated to learn this little fact in an article about printing and typesetting by Carolyn Roberts in The Magazine. (via pjohnkeane)
